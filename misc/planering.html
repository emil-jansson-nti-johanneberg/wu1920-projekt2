Idag gjorde jag klart index sidan för mobilen. Ska först göra hemsidan för mobilen och sen anpassa som en webbsida. Började med att lära mig hur man gör en hamburgermeny (expanding). Gjorde headern (med bilden) responsive och gjorde även en responsive hamburgermenu med lite javascript i. Det som gick bra idag var att jag skissade klart index sidan för mobilen, detta gjorde så att jag kunde fokusera på att "tillverka" hemsidan och börja med en navbar som ska vara med i varje page (html fil). Något som gick dåligt var att jag glömde av att länka till javascripten längst ner i html filen, vilket tog tid att fixa. Nästa lektion vill jag påbörja med footern och index för mobilen.
